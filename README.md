# Hi, I'm José Manuel Echevarría Rubio

**Oceanographer & Geospatial Scientist** | PhD Candidate in Marine Sciences

I develop machine learning pipelines for satellite remote sensing of marine
ecosystems and build ocean hydrodynamic models for ecological connectivity
studies.

## Current Research

- **Kelp Forest Dynamics** — Detecting *Nereocystis luetkeana* and
  *Macrocystis pyrifera* from Sentinel-2 imagery using XGBoost, KNN,
  Random Forest, and 1D-CNN across the Northeast Pacific (2015–2025)
- **Kelp Spore Dispersal Modeling** — FVCOM hydrodynamic model
  (804K-node unstructured mesh) + OpenDrift/OceanParcels Lagrangian
  particle tracking for population connectivity in British Columbia
- **Sargassum Detection** — ML classification of pelagic Sargassum
  in the Caribbean Sea and Equatorial Atlantic from MODIS and Landsat-8
